Note

  • The method used to sort the energy array is non-standard, but is much more easier to implement and faster to run:
    i) The lowest energy value is found on array A and saved at first position of array B; ii) In array A, such position is replaced by INFINITY;
    iii) Now the lowest energy value from array A will be the second value from array B and so on...
